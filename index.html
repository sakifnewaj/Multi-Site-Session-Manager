Troubleshooting Login

Problem: "It loads the login page, but doesn't redirect after login."

This is caused by browser Third-Party Cookie Blocking. The cPanel session cookie cannot be set inside the iframe.

Solution 1: The "Magic Init"

On the session frame, click the Init button. This opens a popup to establish the session, then automatically reloads the iframe.

Solution 2: Pop-out Mode

Some servers (like WHM/cPanel) strictly forbid iframes (`X-Frame-Options: DENY`). Use the button to open it in a separate window.
